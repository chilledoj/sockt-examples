# SOCKT Examples

This repo shows some examples of how to use the [sockt](https://github.com/chilledoj/sockt) library.

## How to Run

**Run Web Server with WS using [coder/websocket](github.com/coder/websocket)**
```shell
go run main.go coderws
```

**Run Client with WS using [coder/websocket](github.com/coder/websocket)**
```shell
go run cmd/client.go coderws
```


## Info

These don't do much but show how things join together.
